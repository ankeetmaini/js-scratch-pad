# JS ScratchPad (an electron app!)

> Because @getify said this https://twitter.com/getify/status/1093578067698700289

A place where you can run JS code (in a new tab via puppeteer), type and when **ready hit Cmd+Enter**

![](JSScratchPad.gif)

### Install

```
npm install
```

### Run

```
npm start
```
